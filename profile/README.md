### Open-Source Tools

| Repository | Description |
|:-----------|:------------|
| [**redhound-arsenal**](https://github.com/redhoundinfosec/redhound-arsenal) | 76 AI-agent security skills for Kali Linux tools — pentest, red team, forensics, OSINT |
